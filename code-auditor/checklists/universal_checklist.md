# 通用安全检查清单

> 不区分语言/框架的通用安全检查项，作为 D1-D10 维度审计的补充。

---

## 基础安全

- [ ] 所有敏感操作需认证
- [ ] 敏感数据（密码/密钥/token）不在日志中输出
- [ ] 错误响应不包含堆栈跟踪/内部路径/SQL 语句
- [ ] 生产环境 Debug/开发模式关闭
- [ ] HTTP 安全头配置（CSP/X-Frame-Options/HSTS）

## 输入输出

- [ ] 所有用户输入有服务端验证
- [ ] 数据库查询使用参数绑定/ORM
- [ ] HTML 输出自动编码（autoescape）
- [ ] 文件路径操作有边界检查
- [ ] 外部命令执行使用数组参数（非字符串）

## 认证授权

- [ ] 密码使用安全哈希存储（bcrypt/argon2）
- [ ] 登录接口有速率限制
- [ ] Session/Token 有合理过期时间
- [ ] 所有写操作验证 CSRF token
- [ ] 资源访问验证归属（防 IDOR）

## 加密通信

- [ ] 敏感数据传输使用 HTTPS/TLS 1.2+
- [ ] 无硬编码密钥/证书
- [ ] SSL/TLS 证书校验未禁用
- [ ] 无弱加密算法（MD5/SHA1/DES）

## 文件上传

- [ ] 文件类型白名单验证
- [ ] 文件大小限制
- [ ] 上传目录不可执行
- [ ] 存储使用随机文件名

## 依赖安全

- [ ] 有依赖锁文件
- [ ] 无已知高危 CVE 依赖
- [ ] 定期更新安全补丁

## 日志审计

- [ ] 关键操作有审计日志
- [ ] 日志不包含敏感信息
- [ ] 日志可防篡改

## 部署配置

- [ ] 默认密码已修改
- [ ] 管理接口不对外暴露
- [ ] 不必要的端口/服务关闭
- [ ] CORS 非 `*`（带 Credentials 时）
