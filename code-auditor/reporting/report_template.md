### 审计方法

- **审计框架**: Code Auditor（EALOC 权重 + 双轨并行）
- **轨道 A（控制建模法）**: 枚举敏感操作，验证安全控制完整性
- **轨道 B（数据流分析）**: 追踪 Source → Sink 路径，验证净化有效性
- **工具链**: scan_sinks.py 粗筛 → LSP/Grep 深度追踪 → 手动验证
